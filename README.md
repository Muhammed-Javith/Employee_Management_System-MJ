# Employee_Management_System-MJ
Employee Management System Project 

## Technologies Used

  - Java
  - Spring Boot 
  - Spring Data JPA (Hibernate)
  - Spring Security
  - MySQL
  - Eclipse STS
  - Maven
  - Tomcat
  - Thymeleaf

## Features

- **List Employees**
- **Add Employee**
- **Update Employee**
- **Delete Employee**
- **Pagination**
- **Sorting**
- **User Login**
- **User Registration**
- **User Logout**
